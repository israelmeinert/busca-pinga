# buscaPinga
buscador de bebidas perto de sua localização e anunciar as promoções.
# Modulos
* Comerciante - Cadastramento de produtos, Anuciar promoção.
* Consumidor  - Busca de produtos, configurar busca automatica, anunciar promoção.
